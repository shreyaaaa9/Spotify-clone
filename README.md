# Spotify-clone
A responsive Spotify-inspired music streaming web application built using HTML and CSS. Features a modern UI with a sidebar navigation menu, music library section, album and playlist cards, and a music player interface, closely replicating the look and feel of Spotify's desktop experience.
Features
🎵 Spotify-inspired user interface
📱 Responsive design
🎨 Modern and clean layout
📂 Sidebar navigation menu
❤️ Music library and playlists section
▶️ Music player controls
🖼️ Album and playlist cards
💻 Built with HTML5 and CSS3
Technologies Used
HTML5
CSS3
Font Awesome
